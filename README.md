# Respiratory_Data

Datafiles Number Order: 

Patient, 
orientation, 
position, 
seconds per inhale/exhale, 
patient state, 
_trial#

Specifics:

	Orientation: 
		0 = horizontal
		1 = vertical

	Position:
		0 = sternum
		1 = centre of abdomen

	Patient state:
		0 = no exercise
		1 = moderate activity
		2 = intesnse activity
		3 = holding breath
